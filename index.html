Welcome to the ViPLab Documentation

Virtual Programming Laboratory

The virtual programming lab ViPLab is a browser-based programming environment. It can be used in ILIAS which enables the execution of programming tasks in online tests. In combination with a management system for research data, like DaRUS, scientists can publicize their code together with their results, for others to try and take a look at.
